# PDFIND

Have you ever ctl-f a keyword in a large PDF only to be overwhelmed with 1000s of results?

Now, its easier!

This extension adds a find menu to PDF.JS for easy browsing of large queries

## The Morzilla PDF.js project

https://github.com/mozilla/pdf.js

PDF.js is a Portable Document Format (PDF) viewer that is built with HTML5.
